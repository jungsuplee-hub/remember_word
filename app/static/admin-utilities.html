한자어 뜻 자동 입력

A열에 한자어가 있는 엑셀 파일을 업로드하면 B열에 뜻(한글)을 채워드립니다.

업로드된 파일의 첫 번째 시트에서 한자어를 찾고 B열이 비어 있는 행에만 뜻을 입력합니다.

결과 파일 다시 다운로드

사용 안내

  • 한자어가 포함된 행만 처리하며, 이미 뜻이 있는 경우에는 값을 덮어쓰지 않습니다.
  • 한자어가 아닌 제목 행은 자동으로 건너뜁니다.
  • 다운로드한 파일을 열어 채워진 뜻을 검토한 후 저장하세요.
