업로드 파일 선택

폴더 · 그룹 · 단어 · 뜻 열을 포함한 파일을 선택하세요.

업로드 설정

엑셀(.xlsx) 또는 CSV 파일을 업로드하면 폴더와 그룹이 자동으로 생성되고 단어가 등록됩니다.

단어 관리로 돌아가기

* 동일한 단어가 존재하면 최신 정보로 업데이트되고, 필수 값이 없는 행은 자동으로 건너뜁니다.

엑셀 샘플 가이드

샘플 이미지를 참고하여 열 이름과 데이터를 준비하세요.

필수 열은 폴더, 그룹, 단어, 뜻입니다. 선택적으로 언어, 별점, 메모 등을 추가할 수 있습니다.

  • 폴더와 그룹이 존재하지 않으면 자동으로 새로 만들어집니다.
  • 기본 언어를 입력하면 새로 생성되는 폴더의 언어로 설정됩니다.
  • 중복 단어는 최신 정보로 업데이트되고, 누락된 데이터가 있는 행은 건너뜁니다.
